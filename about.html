About BlancoSqlEditorPlugin (Blanco SQL Editor Plugin)

Aug 16, 2005

BlancoSqlEditorPlugin is a free SQL Formatter for Eclipse plug-in.

  1. BlancoSqlEditorPlugin is associated to *.sql file type. It works as SQL Text Editor with SQL syntax coloring.
  2. BlancoSqlEditorPlugin add popup menu 'Format as SQL' for #TextEditorContext.

Blanco SQL Editor Plugin (BlancoSqlEditorPlugin) is part of Blanco Framework.
